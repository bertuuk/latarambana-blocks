== Description ==

Marketing Blocks by Bertuuk adds custom Gutenberg blocks focused on marketing and email integration. Includes a GetResponse form block to easily embed forms in your pages or posts.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/bertuuk-marketing-blocks` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate through the 'Plugins' screen in WordPress.
3. Add the GetResponse Form block via the block editor.

== Frequently Asked Questions ==

= Does it require a GetResponse account? =
Yes. You need an active GetResponse account to use the form block.
